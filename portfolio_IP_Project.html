Portfolio Website
ASP.NET

Academic Project, CSE 686: Internet Programming
Internet Programming: HTML5, CSS3, JavaScript, ASP.NET core MVC, ASP.NET Razer, and React.

As a software engineer, a person develops and work on multiple projects and have a portfolio always help to display one's interest and showcase all the skills.
Thus I decided to build a portfolio website using HTML5, CSS3, JavaScript, ASP.NET Core MVC. I implemented the ability for Admin to login and edit content and visitors to be able to browse and fill and submit the form to reach out to an admin.
I followed the waterfall SDLC methodology and implemented functionality to upload, edit, and delete media content using ASP.NET web API.
In the end, I demonstrated a portfolio website with multiple pages and custom styling without bootstrap, and the ability to submit a form which was emailed to the admin.
