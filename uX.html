The whole idea behind the uX project is to provide localized thermal management for office workers, which would result in a dramatic reduction in building energy use and a more comfortable environment to work in and be more productive.
I joined the team with the responsibility to migrate uX to IoT architecture and design, develop and deploy an App on Rpie 3 to control uX with the ability of data acquisition and logging.
I conducted my research on how I could best use the tools and resources provided and developed an UWp (universal Windows Program) application to be deployed on Rpie 3 (Raspberry Pi 3) running windows IoT core operating system. The front end was developed using XAML and backend was developed using C#, SQL, and Azure IoT Central. I also incorporated multiple data logging medium and calibrated multiple sensors.
After multiple iterations of app development in an agile environment, which included a good amount of troubleshooting, debugging and optimization I completed the working prototype.
